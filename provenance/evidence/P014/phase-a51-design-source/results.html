Distance heatmap

Each square reports the Classic Delta distance between two documented texts. Lighter values are smaller within this selected matrix.

The white diagonal is each text compared with itself, which is always distance zero. Colours are scaled inside this matrix only: the same colour can mean a different value in another comparison or another run. For numeric comparison, use the distance matrix below.

Method key

Most frequent words (MFW)
Only the most common words shared across the corpus are compared. This specimen uses the fixture's small feature set.
Classic Delta
Classic Delta standardizes word frequencies and measures the average distance between texts. A smaller distance means relatively greater stylistic proximity inside this corpus; it does not prove authorship.
Reading a distance
Smaller means relatively nearer within this corpus and this parameter cell. Values are not percentages or probabilities.
Corpus relativity
Every number depends on which texts are in the corpus. Adding or removing a text changes all distances.
No universal threshold
There is no fixed value that separates "close" from "far". Compare values against each other, not against a rule of thumb.

Nearest neighbours, including ties

How to read this map

The matrix is authoritative
The distance matrix above is the evidence. This map is a two-dimensional approximation of it.
Apparent clusters can be artefacts
Groupings that appear in two dimensions may be projection artefacts. Verify any pattern against the matrix values before interpreting it.
Axes carry no meaning
Classical multidimensional scaling places texts so that their spacing approximates the selected distance matrix in two dimensions. The axes have no intrinsic literary meaning.

Not an author identification
