# P017 Visual Diagnosis of the Live P016 Page (screenshot-based, 2026-07-21)

Evidence: `provenance/evidence/P017/before/` (live captures, 1440×1000 and 390×844).
Scope: visual/experiential quality only. Scientific copy, semantics, a11y and
performance previously verified and NOT in question here.

## Findings (blocking for the P017 ambition level)

1. **No visual protagonist.** In `p016-live-1440-hero.png` the right ~56% of the
   first viewport is effectively empty: the "drift field" renders as faint 1px
   dashes and dots at very low opacity on the dark band. The caption
   ("Conceptual illustration · not an analysis result") annotates an
   illustration that is barely perceptible. First-viewport impact relies
   entirely on one Inter headline.
2. **One repeated layout.** S2–S7 alternate between a 5/7 two-column split and
   a centered-heading + card-grid pattern. `p016-live-1440-scroll33.png` shows
   the archetype: centered H2, one paragraph, four identical bordered white
   cards. The same skeleton recurs for corpus ledger, MFW lenses, mini-lenses
   and path cards — a component-library rhythm, not an art-directed sequence.
3. **Cards as the only visual container.** Every semantic object (document,
   lens, run record, path) is expressed as the same rounded bordered card.
4. **Large dead bands.** Full-width ivory strips with no compositional content
   separate sections; they read as unfinished space, not designed pause.
5. **Typography without identity.** Inter + Source Sans at large sizes is
   clean but characterless; no display face, no cropping, no scale drama, no
   editorial asymmetry. The page could belong to any careful SaaS product
   after a logo swap — the exact failure condition named in the brief.
6. **One motion verb.** Everything enters with the same fade-up; the progress
   hairline and header state are cosmetic. Scroll reveals content but never
   transforms anything; there is no continuity of a single object across
   sections, no arc, no payoff. The S5 schematic stack reads as documentation
   figures.
7. **Mobile is a stacked reduction.** 390px keeps the same blocks in one
   column; nothing is recomposed for the format.

## What survives into P017 unchanged

Verified claims and boundary sentences; product boundaries (stylometry, not
topic modelling / semantic AI / detection); equal-MFW discipline;
matrix-primary hierarchy; "Public alpha · Experimental" status; CTA
destination; WCAG-level semantics; no fake results anywhere.

## Verdict

P016 is a competent, honest academic product page and a weak cinematic
narrative. The visual layer is rebuilt from scratch in P017; nothing in this
diagnosis is fixable by polishing the existing composition.
