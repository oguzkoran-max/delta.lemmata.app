# P009 Start Prompt: Results, Explanations, and Interpretive Guardrails

## Instruction

Continue Delta from the exact P008 minimum-alpha evidence commit
`e2fb35ece5eb0db3eec16486a1960d92985fc2b1` on branch
`codex/p009-results`. Build the minimum P009 public-alpha slice schema-first and
tests-first. Turn one guardian-confirmed P006 result into a readable,
downloadable result view without changing P006 mathematics, hiding failed cells,
or producing automatic literary interpretation.

## Result Boundary

- P006 `stylo-worker-result-v1` is the sole numerical source.
- Re-read the retained result only through the existing owned capability and
  verify its byte count, SHA-256, canonical form, schema, request ID, outcome,
  and guardian confirmation before projection.
- Keep all four declared Guided cells visible: 100, 300, 500, and 1,000 MFW.
- The 500-MFW cell opens first as a fixed reading reference, never as an optimum.
- Cell inspection changes the visible view only. It never reruns, selects, drops,
  or ranks a parameter setting after seeing the result.
- A failed or `not_enough_features` cell remains explicit and cannot silently
  inherit another cell's matrix.

## Minimum Public-Alpha Output

1. A cell-status overview for the complete four-cell run.
2. A selectable complete-cell detail view, defaulting to the 500-MFW reference
   when it is available.
3. A distance-matrix heatmap backed by an exact semantic table.
4. A nearest-neighbour table that preserves exact ties.
5. A deterministic work-level classical MDS map backed by a coordinate table.
6. A strict result-view JSON download containing no raw text, token stream,
   selected feature words, server path, capability, secret, or private workspace
   identifier.
7. `What this shows` and `What this does not show` beside every analytical panel.

## Interpretation Boundary

- Smaller Delta distance means relative proximity only within this corpus and
  parameter cell.
- Nearest neighbour does not identify an author or establish influence,
  authenticity, intention, chronology, quality, or causation.
- MDS axes have no intrinsic literary meaning; orientation is arbitrary and a
  two-dimensional map may approximate, not reproduce, all distances.
- No confidence, probability of authorship, proof, pure style, ageing,
  maturation, turning-point, or causal language may appear.
- Unknown labels remain `unknown_holdout`; all-known labels remain
  `transductive_exploratory`.

## UX and Accessibility Boundary

- Keep the interface English-only for v0.1 and use the restrained Lemmata-family
  mint, teal, coral, blue, and neutral palette.
- Charts must be nonblank, responsive, keyboard-adjacent, and paired with
  semantic tables containing the same labels and values.
- At 1280, 390, and 320 CSS-pixel widths, controls, labels, legends, and tables
  must not overlap or create page-level horizontal overflow.
- Never render raw text, feature words, capabilities, paths, or opaque ownership
  material in the browser, logs, downloads, screenshots, or provenance.

## Stop Conditions

Stop and record the blocker instead of improvising when:

- P006 output lacks a value needed by a proposed chart;
- a result cannot be reauthorized and verified before projection;
- an export would precede successful input/work cleanup;
- a visual cannot maintain exact semantic-table parity;
- a design would invite post-result parameter cherry-picking or a prohibited
  scientific claim;
- the public-alpha date would require weakening P005-P008 privacy, validation,
  lifecycle, or evidence gates.
